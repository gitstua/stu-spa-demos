From Kingsley's To Casey

A deliberately overbranded parody landing page for the shortest serious race on the calendar: a straight shot from Kingsley's Chicken to Casey Jones Pub.

4PM Sunday Start

The main event is scheduled for 4PM Sunday, which feels late enough to be athletic and early enough to still make the pub.

10m In Centimetres

The headline route is set to 1,000 cm, because converting 10 m into centimetres is objectively the right amount of unnecessary specificity.

Hash-Powered State

The selected route and open accordion panel live in the URL fragment, so the mockup stays shareable without cookies, localStorage, or backend state.

What This Page Does

This standalone SPA mocks the marathon-event page from the supplied screenshot, but rebrands it as the Casey Jones Marathon. The default course runs from Kingsley's Chicken to Casey Jones Pub, displays the 10 metre route as 1,000 centimetres, and calls out the main event start time of 4PM Sunday.

How It Works

The page renders a poster-style event card, an accordion menu, and a cinematic hero panel that echoes the original composition without needing a build step or external framework. Selecting a route updates the main copy and the URL hash, while social metadata in the head gives WhatsApp and other apps something useful to preview.

State Model

The hash stores two values: the current distance option and the open accordion panel. That keeps the mockup linkable and lets the browser restore the selected view on refresh.
