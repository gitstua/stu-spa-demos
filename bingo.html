This Single Page Application (SPA) allows you to create and manage a bingo card. You can toggle edit mode to change the text of each cell, and your bingo card is stored in the URL fragment, so you can bookmark it to save your progress. In edit mode, you can click on any cell to enter text. When not in edit mode, clicking a cell will mark it as crossed. Use the "Copy URL" button to copy the current state of your bingo card to the clipboard, so you can share it or save it for later. If you lose the URL, you'll get a fresh card next time.
